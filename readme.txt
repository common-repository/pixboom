== Description ==

Pixboom is a simple plugin that allows you to quickly and easily tag your images with text and/or clickable links. Any publisher can use this to add more information to any picture. It can be used to show where to find the products on display, what people are where in a picture or just to point out a certain detail of an image and explain it to the viewer.

== Installation ==

Installation is simple:

1. Upload the Pixboom files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click 'Pixboom' in the 'Settings' menu in WordPress and fill in your secret identifier. If you don't have one yet, visit http://www.pixboom.com to sign up.
